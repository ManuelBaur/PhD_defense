\begin{textblock}{0.5}(0.05,0.15)

Generalized Beer-Lambert
\[ I(x) = I_0 \exp(-\mu_\text{eff} (x) \, x) \]

Model function
\[ \mu_\text{eff} (x) = a + \frac{b}{x^\alpha} \]\\[0.5cm]
}

\visible<2->{
Solve
\[ a x + b x^{1-\alpha} + \ln \left(\frac{I(x)}{I_0}\right) = 0\]
e.g.\ Newton's method or look-up table}
\end{textblock}
